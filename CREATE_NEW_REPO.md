# Create New GitLab Repository for Mobile App

## Step 1: Create New Repository on GitLab

1. Go to your GitLab group: https://gitlab.com/degirmenjian.hagop1998-group
2. Click **"New project"** or **"Create project"**
3. Choose **"Create blank project"**
4. Fill in:
   - **Project name**: `ISS-Mobile`
   - **Project slug**: `ISS-Mobile` (auto-filled)
   - **Visibility**: Private (recommended)
   - **Initialize with README**: ‚ùå **Leave unchecked**
5. Click **"Create project"**

## Step 2: After Creating the Repository

Once you create the repository, GitLab will show you the repository URL. It will be:
- `https://gitlab.com/degirmenjian.hagop1998-group/ISS-Mobile.git`

**Share this URL with me, or I can help you update the remote once you create it!**

## Step 3: Update Remote and Push

After you create the repository, I'll help you:
1. Remove the old remote
2. Add the new remote pointing to ISS-Mobile
3. Push your mobile app code to the new repository

This will give you a clean, separate repository just for your mobile app!
